About This System

NewsGuard AI is a machine learning web application designed to detect whether a news article is REAL or FAKE.

Technologies Used:
  • Python
  • Flask Web Framework
  • TF-IDF Vectorization
  • Logistic Regression
  • Scikit-learn

The model was trained on a large Kaggle dataset containing tens of thousands of real and fake news articles, achieving high classification accuracy.
